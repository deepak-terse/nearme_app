## Prerequisites
*  Flutter
*  Dart
*  Android Basics (optional)

## Build Setup

``` bash
# clone the project
git clone https://github.com/deepak-terse/nearme-app.git
cd nearme-app

# install dependencies
flutter pub get

# run the application
flutter run

# hot reload
Press r after flutter run

# build a release version
flutter clean
flutter build apk --target-platform=android-arm64
```
